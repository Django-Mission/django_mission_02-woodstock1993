### 실행방법
- git clone 뒤 가상환경으로 들어가서 Challenge 폴더가 있는 경로에서 python manage.py runserver로 서버 실행
- 홈주소: local/
- 문의주소: local/support/faq/
- 답변주소: local/support/answer/
